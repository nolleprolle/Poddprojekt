# Poddprojekt

> Utvecklad av **Orumatic DevLab** för kunden **Smultronmos AB**  
> En prototyp för att hantera och organisera information om poddar via RSS-flöden.  
> (Applikationen spelar inte upp ljud, utan fungerar som ett smart register över poddar och avsnitt.)

## Projektöversikt
Applikationen gör det möjligt för användare att:
- Hämta och läsa information om poddar via RSS-länkar  
- Spara valda poddar i sitt eget register  
- Visa titel, beskrivning och publiceringsdatum för avsnitt  
- Kategorisera och organisera sina poddar  
- Spara data i molnet via **MongoDB Atlas**
